# KOLLAB Home Page

This project now includes a warm, community-driven company profile experience for KOLLAB built with Vite, React, TypeScript, Tailwind CSS, and GSAP.

## Run locally

```bash
npm install
npm run dev
```

The app will be available at http://localhost:5173.

## Build and test

```bash
npm run build
npm run test
```

## Update content

- Edit the home copy in [src/data/home.json](src/data/home.json)
- Update hero visuals in [src/components/Hero/index.tsx](src/components/Hero/index.tsx)
- Adjust brand colors in [tailwind.config.ts](tailwind.config.ts)
